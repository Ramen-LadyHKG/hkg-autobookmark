# hkg-autobookmark
利用高登自帶嘅bookmark功能，自動bookmark留言過嘅post

## 使用限制
高登新網頁 [https://forum.hkgolden.com/user/xxxxxxx/posthistory](https://forum.hkgolden.com/user/xxxxxxx/posthistory)

## 使用方式
1) 入去你嘅個人起底
2) F12去console
3) 將個hkg-autobookmark.js嘅code貼上並執行
